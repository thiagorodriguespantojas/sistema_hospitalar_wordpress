# Glossário de Termos Técnicos do Hospital Management System

Este glossário fornece definições para termos técnicos comumente usados no Hospital Management System.

## A

**API (Application Programming Interface)**: Um conjunto de protocolos e ferramentas para construir aplicações de software.

**Autenticação de Dois Fatores (2FA)**: Um método de segurança que requer dois tipos diferentes de identificação para acessar uma conta.

## B

**Backup**: Uma cópia de dados armazenada separadamente do original, usada para restauração em caso de perda de dados.

**Banco de Dados**: Um sistema organizado para armazenar, gerenciar e recuperar informações.

## C

**Cache**: Armazenamento temporário de dados para acesso rápido.

**Criptografia**: O processo de codificar informações de forma que apenas partes autorizadas possam acessá-las.

## D

**Dashboard**: Uma interface visual que exibe informações importantes de forma consolidada.

**DICOM (Digital Imaging and Communications in Medicine)**: Um padrão para manipulação, armazenamento, impressão e transmissão de imagens médicas.

## E

**EHR (Electronic Health Record)**: Um registro digital sistemático de informações de saúde de pacientes.

**Endpoint**: Um ponto final de comunicação em uma API ou sistema de rede.

## F

**Firewall**: Um sistema de segurança que monitora e controla o tráfego de rede de entrada e saída.

## G

**GDPR (General Data Protection Regulation)**: Regulamento da União Europeia sobre proteção de dados e privacidade.

## H

**HIPAA (Health Insurance Portability and Accountability Act)**: Lei dos EUA que fornece padrões de segurança para proteger informações de saúde dos pacientes.

## I

**Integração**: O processo de combinar diferentes subsistemas ou componentes em um único sistema.

## J

**JWT (JSON Web Token)**: Um padrão aberto que define uma forma compacta e autônoma de transmitir informações com segurança entre as partes como um objeto JSON.

## L

**LGPD (Lei Geral de Proteção de Dados)**: Lei brasileira que regula o tratamento de dados pessoais.

**Log**: Um registro de eventos ocorridos em um sistema de computador.

## M

**Middleware**: Software que atua como uma ponte entre um sistema operacional ou banco de dados e aplicativos.

## P

**PHI (Protected Health Information)**: Qualquer informação sobre saúde que possa ser vinculada a um indivíduo específico.

## R

**RBAC (Role-Based Access Control)**: Um método de regular o acesso a recursos de computador ou rede com base nas funções de usuários individuais.

**REST (Representational State Transfer)**: Um estilo de arquitetura de software para sistemas distribuídos, como a World Wide Web.

## S

**SSL/TLS**: Protocolos criptográficos projetados para fornecer comunicações seguras pela Internet.

## T

**Telemedicina**: A prática de cuidar de pacientes remotamente quando o provedor e o paciente não estão fisicamente presentes um com o outro.

## V

**VPN (Virtual Private Network)**: Uma conexão criptografada pela internet de um dispositivo para uma rede.

## W

**WebSocket**: Um protocolo de comunicação que fornece canais de comunicação full-duplex sobre uma única conexão TCP.

Este glossário será atualizado regularmente para incluir novos termos e tecnologias relevantes para o Hospital Management System.
